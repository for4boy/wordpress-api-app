# 零股之易 - 自動化 WordPress API 應用

## 📌 應用簡介
這是一個基於 WordPress API 的自動化應用，旨在自動發佈「零股之易」相關的部落格文章，讓小資金投資者輕鬆參與市場。

## 🚀 功能特色
- 透過 WordPress API 自動發佈文章。
- 支援批次發佈與內容管理。
- 使用 Python 實現簡單的自動化。

## 📄 使用方式
- 取得 WordPress API 存取 Token（OAuth 2.0）。
- 將 Python 腳本設定為自動化發佈工具。
- 可擴充至多網站發佈。
